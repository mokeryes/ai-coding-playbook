# 任务收尾汇报格式

> 阶段 1 占位文件。**阶段 2 会从 wework-opus `CLAUDE.md` § 16 抽取并扩充**。

## 触发时机

每个独立任务交付时(代码写完、文档改完、配置改完之后)。

## 6 项汇报(简洁,无关项可省略)

1. **完成了什么**(1-2 句)
2. **修改了哪些文件**(列表,带路径)
3. **新增了哪些测试 / 验证**(如适用)
4. **AI 自跑了哪些验证**(`py_compile` / `swift build` / `tsc --noEmit` / ...)
5. **哪些内容未验证**,需要用户跑(给出明确命令)
6. **用户下一步建议执行的命令**

不写长篇解释,除非用户问"为什么"。
