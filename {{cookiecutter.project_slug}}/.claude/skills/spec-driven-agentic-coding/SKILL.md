# Spec-Driven Agentic Coding(占位)

> 阶段 1 占位文件。**阶段 2 会用 wework-opus `SPEC_DRIVEN_AGENTIC_CODING_SKILL.md` 完整内容替换**。

核心思想:

```
先定规格,再写代码;
边写边测,提交留痕;
AI 执行,人来把关。
```

## 总流程(8 步)

1. 需求对齐
2. 编写或更新规格文档
3. 等待明确实现指令
4. Agentic 实现
5. 同步补测试
6. 验证
7. 人审 diff
8. Git 提交
